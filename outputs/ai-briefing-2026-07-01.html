米政権、Claude Fable 5 / Mythos 5 の輸出規制を全面解除——19日間の停止から本日復旧

Anthropic は6月30日（現地時間火曜夜）、米商務省が Claude Fable 5 と Claude Mythos 5 への輸出規制を全面解除したと発表した。両モデルは6月9日にリリースされたが、信頼パートナー（Amazon と報じられる）がガードレールを回避するジェイルブレイク手法を発見したことを受け、6月12日に商務省が外国人ユーザーを含む全世界での利用停止を命じていた。19日間の停止を経て、アクセスは本日（7月1日水曜）から段階的に復旧する。 商務長官ハワード・ルトニック氏はSNSで解除を発表し、Anthropic は「明日から復旧を開始する」との声明を出した。今回の一連の措置は、先端AIモデルに国家安全保障の観点から輸出管理が適用された初のケースとして注目されている。

📎 出典： CNBC — Anthropic Export Controls Lifted ／ Anthropic 公式声明 ／ CNN Business
商用フロンティアAIが国家の輸出管理下に置かれ、しかも3週間足らずで解除されるという展開は前例がない。ジェイルブレイク耐性の脆さが露呈した以上、次のモデル世代では「政府審査を前提としたリリース設計」が業界標準になる可能性が高い。
高
Amazon AWS、10億ドル規模「フォワード・デプロイド・エンジニアリング」新組織を設立

Amazon Web Services（AWS）は6月30日、自社資金 10億ドル を投じて「フォワード・デプロイド・エンジニア（FDE）」専門組織を新設したと発表した。AWSのフロンティアAIエンジニアリング担当VP、フランチェスカ・バスケス氏によると、数千人規模のFDEを配置し、企業の現場に小規模チームを約45日間常駐させて、顧客専用のエージェントAIシステムを構築する。 OpenAI や Anthropic がすでに外部投資家やコンサルティングパートナーと組んで同種の展開部隊を作っているのに対し、AWS は他社と組まず自己資金のみで運営する点が特徴。顧客は新しいAIシステムと同時に、自社で運用を継続できる技術力も獲得できるという。

📎 出典： CNBC — AWS Forward-Deployed Engineers ／ TechCrunch
「フォワード・デプロイド・エンジニア」というPalantir発祥の職種が、今やAI大手3社共通の主戦場になっている。企業向けAI導入は汎用モデルの提供だけでは完結せず、業務に合わせて人手で仕上げる工程が競争優位の分かれ目になっている点が読み取れる。

中
GitHub Copilot、従量課金の初回フル課金サイクルが終了——利用者の請求額が最大50倍に急増

GitHub Copilot は6月1日にサブスクリプション型からトークン従量課金の「AI Credits」制へ移行したが、6月30日に最初の完全な30日間課金サイクルが終了し、実際の請求額が明らかになった。エージェント的な使い方をする開発者からは、月額29ドルの請求が750ドルへ、50ドルが3,000ドルへ跳ね上がったとの報告が相次いでいる。 入力・出力・キャッシュされたトークンすべてに課金される仕組みのため、AIエージェントに複雑な作業を任せるほどコストが線形以上に増大する。上限を自分で設定しない限りブレーキがかからない設計が、今回の「請求書ショック」の直接的な原因とされる。

📎 出典： Tech Times — Copilot Billing Shock Confirmed ／ GitHub 公式ブログ
AIコーディングエージェントの「使い方」自体が変わらないと、従量課金モデルとは相性が悪いことが露呈した。トークン消費量を可視化し予算上限を明示的に設計する運用スキルが、今後は開発チームに必須のノウハウになる。
中
Proton、プライバシー重視AIチャット「Lumo 2.0」を公開——画像生成・暗号化メモリ・Thinkingモードを追加

スイスのプライバシー企業 Proton は6月30日、AIチャットアシスタント「Lumo」を大幅刷新した「Lumo 2.0」を発表した。新機能として画像生成・画像認識、ユーザーが管理する暗号化メモリ、特定用途向けにカスタマイズできる「Custom Lumos」、引用付きのライブWeb検索を追加。応答速度を優先する「Fast」モードと複数ステップの推論を行う「Thinking」モードを切り替えられる新アーキテクチャを採用した。 Proton によると、Lumo 2.0 Lite は旧バージョン比で Artificial Analysis Intelligence Index が127%向上、上位モデルの Lumo 2.0 Max は240%向上したとしている。ゼロアクセス暗号化を維持したまま高性能化を図った点が特徴。

📎 出典： TechCrunch — Lumo Gets an Upgrade ／ 9to5Mac
「プライバシー」と「高性能マルチモーダル」は二択だと思われがちだが、Lumo 2.0 はその折衷案を提示している。企業の機密情報を扱う業務でAIチャットを使いたいが外部送信を避けたい、というニーズに応える選択肢が増えたのは歓迎したい動き。
中
暗号資産取引所OKX、「AIエージェント同士が雇用・支払いを行う」新マーケットプレイス「OKX AI」を開設

暗号資産取引所 OKX は6月30日、クローズドベータを経て「OKX AI」を開発者向けに一般公開した。これはAIエージェントが互いを「雇用」し、業務完了後に自律的にオンチェーンで支払いを行い、実績に基づく「ポータブルな信用（レピュテーション）」を積み上げられる市場だという。人手を介さずAIエージェント間で発注・実行・決済・評価が完結する仕組みを目指している。 同社は、AIエージェントが自律的に経済活動を行う「エージェントエコノミー（Agentic Commerce）」という概念の実装例としてこのサービスを位置づけている。暗号資産の決済インフラとAIエージェントの自律実行を組み合わせた新しい応用領域として注目される。

📎 出典： TechCrunch — OKX Wants AI Agents to Hire and Pay Each Other
「AIエージェントが発注者にも受注者にもなる」市場は新概念の応用先として興味深い一方、詐欺・誤発注・無限ループ的な支払いなど、人間の監督が及ばないリスクへの備えが未成熟なまま先行している印象も強い。ガバナンス設計の検証を注視したい。

Agility Robotics、SPAC上場を通じ米国初の人型ロボット専業上場企業に——新型「Digit v5」も発表

人型ロボット企業 Agility Robotics は、Churchill Capital Corp XI とのSPAC（特別買収目的会社）合併により、企業価値 25億ドル でナスダック上場（ティッカー「AGLT」予定）することを明らかにした。追加で最大2億ドルのPIPE資金を含め、合計6.2億ドル超を新型モデル「Digit v5」の開発と量産拡大に充てる計画。 Digit v5 は最大50ポンド（約22.7kg、v4比40%増）の可搬重量、約22時間の連続稼働、身長7.2フィートを実現し、「人と協働できる安全性を持つ世界初のAI搭載ヒューマノイド」と位置づけられている。既存モデルは Schaeffler・GXO・トヨタ・Mercado Libre など9拠点で6.5万時間超の稼働実績を積んでおり、Digit v5には既に3億ドル超の複数年受注が入っている。

📎 出典： Tech Times — Agility Robotics SPAC Deal ／ The Robot Report
ヒューマノイド専業企業が上場市場に到達したことで、投資判断の材料が「デモの完成度」から「稼働時間・受注額・財務指標」という事業会社としての実績に移っていく。alphaktとしても業界の成熟度を測る定点として追跡する価値がある。
